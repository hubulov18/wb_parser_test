

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run docker container

```bash

$ docker compose up

```
Необходимо добавить credentials.json в корень проекта чтобы можно было работать с гугл таблицами
